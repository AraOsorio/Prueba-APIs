# Conversor de Monedas 💱

## Descripción

Este proyecto es un conversor de monedas desarrollado con **HTML, CSS y JavaScript**.  
Permite ingresar un monto en **pesos chilenos (CLP)** y convertirlo a otra moneda utilizando datos reales obtenidos desde la API pública de **mindicador.cl**.

Además de mostrar el resultado de la conversión, el sistema genera un **gráfico con el historial de los últimos 10 días** del valor de la moneda seleccionada usando la librería **Chart.js**.

Este proyecto fue realizado como parte de una prueba práctica para aplicar el uso de **fetch, consumo de APIs y visualización de datos en JavaScript**.

---

## Tecnologías utilizadas

- HTML  
- CSS  
- JavaScript  
- API: https://mindicador.cl/api  
- Chart.js  

---

## Funcionalidades

- Ingresar un monto en pesos chilenos  
- Seleccionar la moneda a convertir (Dólar o Euro)  
- Consultar datos desde una API usando **fetch**  
- Calcular automáticamente la conversión  
- Mostrar el resultado en el DOM  
- Generar un gráfico con el historial de los últimos 10 días  
- Manejar errores usando **try / catch**

---

## Cómo usar el proyecto

1. Ingresar un monto en pesos chilenos  
2. Seleccionar la moneda que se quiere convertir  
3. Hacer clic en el botón **Buscar**  
4. El sistema mostrará el resultado de la conversión y el gráfico con el historial de la moneda seleccionada
